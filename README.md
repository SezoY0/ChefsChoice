# Chef's Choice Webshop

Dit is een simpele webshop voor Project 4 - Chef's Choice.

## Wat zit erin?
- Homepage
- Gerechtenpagina
- Zoeken en filteren
- Gerecht detailpagina
- Registreren
- Inloggen
- Winkelwagen
- Aantal aanpassen
- Gerecht verwijderen
- Kortingscode systeem
- Bestelling opslaan in database
- Responsive design voor mobiel, tablet en desktop

## Installeren in XAMPP
1. Zet de map `chefs_choice_webshop` in `htdocs`.
2. Open phpMyAdmin.
3. Importeer `database.sql`.
4. Controleer in `config.php` of database naam en wachtwoord goed zijn.
5. Open in je browser: `http://localhost/chefs_choice_webshop/index.php`

## Kortingscodes
- CHEF10 = 10% korting
- WELKOM5 = 5 euro korting
- STUDENT15 = 15% korting
